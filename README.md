## Testing 

You can test it using the following command while in the directory :

Update librairy : 
```bash
 npm install 
```
or : 
```bash
npm update 
 ```

Server launch :
```bash
npm run dev  
 ```

Once the application is launched, you can access it through any search engine parsing the following :

http://localhost:3000
